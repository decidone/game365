홈페이지 제작 목적


 - 게임 배포(일반 사용자용 메뉴)
	- 최신 게임, 인기 게임 리스트(첫 페이지)
	- 검색(장르, 제작자, 제목)
		- 클릭 시 자세한 게임 정보
			- 후원하기(구매하기)
			- 프로젝트 참여 요청
			- 소개글
			- 평가
 - 공동 게임 제작
	- 프로젝트 생성
		-제목, 내용, 장르 등 입력하는 페이지
	- 게임 소개
		-게임 소개 관리 페이지
	- 프로젝트 내 채팅
	- 프로젝트 파일 공유 시스템(git)
	- 수익 설정
		- 펀딩
			- 목표금액 설정
			- 펀딩 내역 관리
		- 몫 분배
			- 배당 설정 페이지
			- 수익 내역 관리
	- 프로젝트 관리
		- 인원 관리
		- 제작 일정 설정
		- 자율 심의


DB
유저
아이디, 비밀번호, 닉네임, 메일주소, 전화번호, 결제 수단, 분야

후원 내역
후원 번호, 유저 아이디, 금액, 후원일

게시글
글 번호, 글 제목, 게시일, 글 내용, 작성자, 장르, 

펀딩

펀딩 내역
펀딩 번호, 유저 아이디, 금액

프로젝트
프로젝트 번호, 프로젝트 이름, 프로젝트 진행 여부(0-제작중, 1- 제작완료, 2- 제작 중단), 펀딩 여부, git 페이지, 심의 여부
